# VSCord

Discord Rich Presence for Visual Studio Code.

## Features

- **Automatic presence updates** - Shows what file you're working on
- **Language-specific icons** - Different icon for each programming language
- **Workspace tracking** - Displays which project you're in
- **Smart idle detection** - Shows AFK status after 2 minutes of inactivity
- **Session timer** - Tracks total coding time per session
- **Manual reconnect** - Click the status bar to reconnect if Discord isn't running

## Usage

Extension and it works automatically and updates when you:
- Switch between files
- Open different projects
- Go AFK

## Requirements

- Discord desktop app must be running
- No configuration needed - works out of the box

## Status Bar

Check the bottom-left status bar for connection status:
- ✓ **Connected** - Working normally
- ✗ **Disconnected** - Click to reconnect

## Privacy

All processing happens locally. No data is sent anywhere except to your local Discord client.
